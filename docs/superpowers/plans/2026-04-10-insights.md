# Insights Feature Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Extract Claude Code insight blocks from conversation JSONL files, persist them to per-project `INSIGHTS.md` files (append-only, latest-first), and expose them through a cross-project browser page and per-project tab with search.

**Architecture:** Insights are extracted from `~/.claude/projects/*/` JSONL conversation files by scanning assistant text blocks for marker patterns (`★`, `✻`, `💡`, `**Insight**`, `## Insight`). A bootstrap script does a one-time import; subsequent scans append incrementally. Dedup uses a content hash stored in HTML comments within `INSIGHTS.md`. The UI follows existing `ManualSteps` and `Sessions` patterns.

**Tech Stack:** Next.js 16 App Router, TypeScript, Tailwind CSS v4, lucide-react icons, existing hand-rolled UI components.

```typescript
import { promises as fs } from "fs";
import path from "path";
import crypto from "crypto";
import { InsightEntry, InsightsInfo } from "../types";

/* eslint-disable @typescript-eslint/no-explicit-any */

/**
 * Generate a stable dedup ID from insight content.
 */
export function insightId(content: string): string {
  return crypto.createHash("sha256").update(content.trim()).digest("hex").slice(0, 12);
}

// ─── Marker patterns (broad set from gist) ──────────────────────────

// Opening markers: backtick-wrapped with star/asterisk + dashes, or bold/heading
const INSIGHT_OPEN_RE = /^[`\s]*[★✻💡]\s*Insight[\s─━\-_]*[`\s]*$|^\*\*Insight\*\*|^##\s+Insight/i;

// Closing marker: backtick + dashes/underscores (the bottom border)
const INSIGHT_CLOSE_RE = /^[`\s]*[─━\-_]{10,}[`\s]*$/;

/**
 * Extract insight blocks from JSONL conversation content.
 * Scans assistant text blocks for insight markers.
 */
export function parseInsightsFromJsonl(
  jsonlContent: string,
  sessionId: string,
  projectSlug: string,
  projectPath: string
): InsightEntry[] {
  const entries: InsightEntry[] = [];
  const lines = jsonlContent.split("\n").filter(Boolean);

  for (const line of lines) {
    let entry: any;
    try {
      entry = JSON.parse(line);
    } catch {
      continue;
    }

    if (entry.type !== "assistant" || !entry.message?.content) continue;
    const timestamp = entry.timestamp || "";

    for (const block of entry.message.content) {
      if (block.type !== "text" || !block.text) continue;

      const textLines = String(block.text).split("\n");
      let capturing = false;
      let contentLines: string[] = [];

      for (const tl of textLines) {
        if (!capturing && INSIGHT_OPEN_RE.test(tl.trim())) {
          capturing = true;
          contentLines = [];
          continue;
        }

        if (capturing) {
          // Check for closing marker or blank line ending the block
          if (INSIGHT_CLOSE_RE.test(tl.trim())) {
            const content = contentLines.join("\n").trim();
            if (content) {
              entries.push({
                id: insightId(content),
                content,
                sessionId,
                date: timestamp,
                project: projectSlug,
                projectPath,
              });
            }
            capturing = false;
            contentLines = [];
            continue;
          }

          // A completely blank line after content also ends the block
          if (tl.trim() === "" && contentLines.length > 0 && contentLines.some(l => l.trim())) {
            const content = contentLines.join("\n").trim();
            if (content) {
              entries.push({
                id: insightId(content),
                content,
                sessionId,
                date: timestamp,
                project: projectSlug,
                projectPath,
              });
            }
            capturing = false;
            contentLines = [];
            continue;
          }

          contentLines.push(tl);
        }
      }

      // Handle unclosed block at end of text
      if (capturing && contentLines.length > 0) {
        const content = contentLines.join("\n").trim();
        if (content) {
          entries.push({
            id: insightId(content),
            content,
            sessionId,
            date: timestamp,
            project: projectSlug,
            projectPath,
          });
        }
      }
    }
  }

  return entries;
}

// ─── INSIGHTS.md parser ─────────────────────────────────────────────

const COMMENT_RE = /^<!--\s*insight:(\w+)\s*\|\s*session:(\S+)\s*\|\s*(.+?)\s*-->$/;

/**
 * Parse an existing INSIGHTS.md file. Returns entries and the set of known IDs.
 */
export function parseInsightsMd(content: string): { info: InsightsInfo; knownIds: Set<string> } {
  const entries: InsightEntry[] = [];
  const knownIds = new Set<string>();
  const lines = content.split(/\r?\n/);

  let currentId = "";
  let currentSessionId = "";
  let currentDate = "";
  let contentLines: string[] = [];
  let inEntry = false;

  for (const line of lines) {
    const commentMatch = line.match(COMMENT_RE);
    if (commentMatch) {
      // Flush previous entry
      if (inEntry && currentId) {
        const text = contentLines.join("\n").trim();
        if (text) {
          entries.push({
            id: currentId,
            content: text,
            sessionId: currentSessionId,
            date: currentDate,
            project: "",  // filled by caller
            projectPath: "", // filled by caller
          });
        }
      }
      currentId = commentMatch[1];
      currentSessionId = commentMatch[2];
      currentDate = commentMatch[3].trim();
      knownIds.add(currentId);
      contentLines = [];
      inEntry = true;
      continue;
    }

    // Skip the ## heading line and --- separators
    if (inEntry) {
      if (line.match(/^##\s+[★✻💡]?\s*Insight/i)) continue;
      if (line.trim() === "---") {
        // End of entry
        const text = contentLines.join("\n").trim();
        if (text && currentId) {
          entries.push({
            id: currentId,
            content: text,
            sessionId: currentSessionId,
            date: currentDate,
            project: "",
            projectPath: "",
          });
        }
        inEntry = false;
        currentId = "";
        contentLines = [];
        continue;
      }
      contentLines.push(line);
    }
  }

  // Flush final entry
  if (inEntry && currentId) {
    const text = contentLines.join("\n").trim();
    if (text) {
      entries.push({
        id: currentId,
        content: text,
        sessionId: currentSessionId,
        date: currentDate,
        project: "",
        projectPath: "",
      });
    }
  }

  return {
    info: { entries, total: entries.length },
    knownIds,
  };
}

// ─── Scanner function (used by orchestrator) ────────────────────────

/**
 * Read INSIGHTS.md from a project root. Returns InsightsInfo or undefined.
 */
export async function scanInsightsMd(
  projectPath: string
): Promise<InsightsInfo | undefined> {
  try {
    const content = await fs.readFile(
      path.join(projectPath, "INSIGHTS.md"),
      "utf-8"
    );
    const { info } = parseInsightsMd(content);
    // Fill in project info
    const slug = path.basename(projectPath).toLowerCase().replace(/[^a-z0-9-]/g, "-");
    for (const entry of info.entries) {
      entry.project = slug;
      entry.projectPath = projectPath;
    }
    return info.total > 0 ? info : undefined;
  } catch {
    return undefined;
  }
}

```typescript
import { promises as fs } from "fs";
import path from "path";
import { InsightEntry } from "./types";
import { parseInsightsMd } from "./scanner/insightsMd";

/**
 * Append new insights to INSIGHTS.md (prepend at top, latest-first).
 * Returns the count of newly appended entries.
 * Idempotent: skips entries whose ID already exists in the file.
 */
export async function appendInsights(
  projectPath: string,
  entries: InsightEntry[]
): Promise<number> {
  if (entries.length === 0) return 0;

  const filePath = path.join(projectPath, "INSIGHTS.md");

  let existingContent = "";
  let knownIds = new Set<string>();
  try {
    existingContent = await fs.readFile(filePath, "utf-8");
    const parsed = parseInsightsMd(existingContent);
    knownIds = parsed.knownIds;
  } catch {
    // File doesn't exist yet — start fresh
  }

  // Filter out duplicates
  const newEntries = entries.filter((e) => !knownIds.has(e.id));
  if (newEntries.length === 0) return 0;

  // Sort new entries by date descending (latest first)
  newEntries.sort((a, b) => {
    const ta = a.date ? new Date(a.date).getTime() : 0;
    const tb = b.date ? new Date(b.date).getTime() : 0;
    return tb - ta;
  });

  // Format new entries as markdown
  const blocks = newEntries.map((e) => {
    const dateStr = e.date ? new Date(e.date).toISOString().slice(0, 19).replace("T", " ") : "unknown";
    return [
      `<!-- insight:${e.id} | session:${e.sessionId} | ${dateStr} -->`,
      `## ★ Insight`,
      e.content,
      "",
      "---",
    ].join("\n");
  });

  const newSection = blocks.join("\n\n") + "\n";

  // Strip existing header if present, prepend new entries, re-add header
  const header = "# Insights\n\n";
  const body = existingContent.startsWith("# Insights")
    ? existingContent.replace(/^# Insights\s*\n*/, "")
    : existingContent;

  const result = header + newSection + "\n" + body;
  await fs.writeFile(filePath, result, "utf-8");

  return newEntries.length;
}

import { promises as fs } from "fs";
import path from "path";
import os from "os";
import { InsightEntry } from "../src/lib/types";
import { parseInsightsFromJsonl } from "../src/lib/scanner/insightsMd";
import { appendInsights } from "../src/lib/insightsWriter";

function decodeDirName(dirName: string): string {
  return dirName.replace(/^([A-Z])-/, "$1:").replace(/-/g, "\\");
}

function toSlug(dirName: string): string {
  const parts = dirName.split("-");
  const meaningful = parts.slice(parts.findIndex((p) => p.length > 1));
  return meaningful.join("-").toLowerCase().replace(/[^a-z0-9-]/g, "-");
}

async function main() {
  const projectsDir = path.join(os.homedir(), ".claude", "projects");
  console.log(`Scanning ${projectsDir} for insights...\n`);

  let dirs: string[];
  try {
    dirs = await fs.readdir(projectsDir);
  } catch {
    console.error("Could not read", projectsDir);
    process.exit(1);
  }

  let totalInsights = 0;
  let totalProjects = 0;

  for (const dir of dirs) {
    const dirPath = path.join(projectsDir, dir);
    const stat = await fs.stat(dirPath);
    if (!stat.isDirectory()) continue;

    const projectPath = decodeDirName(dir);
    const projectSlug = toSlug(dir);

    // Verify project exists on disk
    try {
      await fs.access(projectPath);
    } catch {
      continue; // Skip non-existent projects
    }

    // Read all JSONL files
    const files = (await fs.readdir(dirPath)).filter((f) => f.endsWith(".jsonl"));
    const allInsights: InsightEntry[] = [];

    for (const file of files) {
      const filePath = path.join(dirPath, file);
      const fstat = await fs.stat(filePath);
      if (fstat.size > 50 * 1024 * 1024) continue; // Skip huge files

      const content = await fs.readFile(filePath, "utf-8");
      const sessionId = path.basename(file, ".jsonl");
      const insights = parseInsightsFromJsonl(content, sessionId, projectSlug, projectPath);
      allInsights.push(...insights);
    }

    if (allInsights.length === 0) continue;

    const appended = await appendInsights(projectPath, allInsights);
    if (appended > 0) {
      console.log(`  ${path.basename(projectPath)}: ${appended} insights imported`);
      totalInsights += appended;
      totalProjects++;
    }
  }

  console.log(`\nDone. ${totalInsights} insights imported across ${totalProjects} projects.`);
}

main().catch((err) => {
  console.error(err);
  process.exit(1);
});
```

import { NextRequest, NextResponse } from "next/server";
import { scanAllProjects } from "@/lib/scanner";
import { getCachedScan, setCachedScan } from "@/lib/cache";
import { InsightEntry } from "@/lib/types";

export async function GET(request: NextRequest) {
  const projectFilter = request.nextUrl.searchParams.get("project");
  const query = request.nextUrl.searchParams.get("q")?.toLowerCase();

  let result = getCachedScan();
  if (!result) {
    result = await scanAllProjects();
    setCachedScan(result);
  }

  let insights: InsightEntry[] = [];

  for (const p of result.projects) {
    if (projectFilter && p.slug !== projectFilter) continue;
    if (p.insights) {
      insights.push(...p.insights.entries);
    }
  }

  // Sort latest-first
  insights.sort((a, b) => {
    const ta = a.date ? new Date(a.date).getTime() : 0;
    const tb = b.date ? new Date(b.date).getTime() : 0;
    return tb - ta;
  });

  // Keyword search
  if (query) {
    insights = insights.filter((i) =>
      i.content.toLowerCase().includes(query)
    );
  }

  return NextResponse.json({ insights, total: insights.length });
}

"use client";

import { useState, useEffect, useCallback } from "react";
import { InsightEntry, InsightsInfo } from "@/lib/types";

interface AllInsightsResult {
  insights: InsightEntry[];
  total: number;
}

export function useAllInsights(projectFilter?: string, query?: string) {
  const [data, setData] = useState<AllInsightsResult>({ insights: [], total: 0 });
  const [loading, setLoading] = useState(true);

  const refresh = useCallback(async () => {
    const params = new URLSearchParams();
    if (projectFilter) params.set("project", projectFilter);
    if (query) params.set("q", query);
    const qs = params.toString();
    try {
      const res = await fetch(`/api/insights${qs ? `?${qs}` : ""}`);
      if (res.ok) setData(await res.json());
    } finally {
      setLoading(false);
    }
  }, [projectFilter, query]);

  useEffect(() => {
    setLoading(true);
    refresh();
  }, [refresh]);

  return { data, loading, refresh };
}

export function useProjectInsights(slug: string) {
  const [data, setData] = useState<InsightsInfo | null>(null);
  const [loading, setLoading] = useState(true);

  const refresh = useCallback(async () => {
    try {
      const res = await fetch(`/api/insights/${slug}`);
      if (res.ok) setData(await res.json());
    } finally {
      setLoading(false);
    }
  }, [slug]);

  useEffect(() => {
    refresh();
  }, [refresh]);

  return { data, loading, refresh };
}

function formatDate(iso?: string): string {
  if (!iso) return "—";

function formatDate(iso?: string): string {
  if (!iso) return "—";
  const d = new Date(iso);
  const now = new Date();
  const diffMs = now.getTime() - d.getTime();
  const diffDays = Math.floor(diffMs / 86_400_000);
  if (diffDays < 1) return "today";
  if (diffDays === 1) return "yesterday";
  if (diffDays < 7) return `${diffDays}d ago`;
  return d.toLocaleDateString();
}

# Insights

The Insights feature extracts and preserves educational insight blocks from Claude Code conversation history.

## How It Works

When Claude Code generates insight blocks (marked with `★ Insight`, `✻`, `💡`, or `**Insight**` headers), they are extracted from conversation JSONL files and saved to `INSIGHTS.md` in each project root.

- **Append-only** — New insights are prepended (latest-first). Existing entries are never modified.
- **Deduplicated** — Each insight is hashed; duplicates are skipped automatically.
- **Persistent** — Insights survive even if conversation history is cleaned up.

## Cross-Project Browser

The `/insights` page shows all insights across your portfolio:

- **Search** — Filter by keyword across all insight content
- **Project filter** — Dropdown to scope to a single project
- **Session links** — Each insight links back to the originating session

## Per-Project Tab

Each project detail page has an **Insights** tab showing insights for that project only, with local search.

## Bootstrap Import

To import insights from existing conversation history:

```
npx tsx scripts/import-insights.ts
```

This scans all `~/.claude/projects/` JSONL files and writes `INSIGHTS.md` to each project that has insights. Safe to run multiple times.

## INSIGHTS.md Format

Insights are stored as markdown with HTML comment metadata:

```markdown
<!-- insight:abc123 | session:xxxxx | 2026-04-08 12:30:00 -->
## ★ Insight
[content]

---
```
```
