# Background Activity

Monitor the background tasks running in Project Minder — scheduled scans, cache updates, and other system activity.

The background activity page shows what's happening behind the scenes as you work: when scans run, when data is refreshed, and the status of long-running operations.

## Activity log

Each line represents a background event with its timestamp, type (scan, cache, etc.), and outcome. Activity streams in real time as Minder performs its background duties.

## Filters and search

Filter the activity log by type to focus on specific kinds of events, and search the activity text to find particular operations or projects.
