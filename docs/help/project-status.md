# Project Status

Every project has a status that helps you organize your work. Statuses are color-coded throughout the dashboard.

## Available Statuses

| Status | Color | Use for |
|--------|-------|---------|
| **Active** | Green | Projects you're currently working on |
| **Paused** | Amber | Projects temporarily on hold |
| **Archived** | Gray | Projects no longer in active development |

## Changing a Project's Status

### From the Detail Page

Open any project and click one of the three status badges (**Active**, **Paused**, **Archived**) near the top. The current status appears highlighted; click a different one to change it.

### Filtering by Status

On the dashboard, use the filter buttons to show only projects with a specific status. See [Search, Filter, and Sort](search-and-filter.md).

## Where Status Appears

- **Project cards** — colored badge in the top-right corner, plus a colored left border
- **Detail page** — status selector in the header
- **Dashboard filters** — count of projects per status

Status changes save automatically and persist between sessions.
