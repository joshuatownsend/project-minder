# Projects

The projects list dashboard displays all projects Minder has scanned in your dev root (`C:\dev\` by default).

Each project card shows at a glance: tech stack, last git commit, port and dev server status, and recent Claude Code activity. Use search and status filters to find the project you're looking for.

## Viewing a project

Click any project card to open its detail page, where you can inspect its context, configurations, recent sessions, and more.

## Starting a dev server

Use the start/stop badge on each card to launch or stop a project's dev server. The port appears inline once the server starts.

## Status and filtering

Filter projects by their current status — active, archived, or excluded from scanning. You can also search by name, language, or tech stack.
