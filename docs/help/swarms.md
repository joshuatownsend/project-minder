# Swarms

Orchestrate multi-agent workflows where several Claude Code agents work in parallel or in sequence to accomplish complex tasks.

Swarms coordinate work across multiple agents — each agent handles a piece of the larger problem and can hand off results to the next agent. Use swarms to parallelize work (batch processing across projects) or chain specialized agents (e.g., an analyzer feeds into a coder feeds into a tester).

## Creating a swarm

Use the Swarm Composer to define your agents, their roles, and how they coordinate. Specify which agents participate, how data flows between them, and what triggers each step.

## Monitoring a swarm

Each swarm shows the status of all its participating agents — running, waiting, or completed. Watch progress in real time and drill into individual agent logs to debug or inspect results.

## Results

Once a swarm finishes, view aggregated results from all agents and export the work they produced.
