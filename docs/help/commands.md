# Commands

Browse and search your Claude Code commands — executable instructions and automations defined across all your projects.

Commands are reusable snippets and workflows stored in project `.claude/commands/` directories. This page aggregates them into one searchable catalog, so you can find and re-run commands from any project without navigating to its directory.

## Finding a command

Use the search bar to find a command by name or description. Filter by project to scope the list to a single project's commands.

## Running a command

Click a command row to view its full body and trigger it. Commands execute in their project directory so file paths and project context are preserved.

## Provenance

Each command shows which project it belongs to and whether it's user-defined, bundled with a plugin, or part of a marketplace extension.
