# Search, Filter, and Sort

## Searching Projects

Type in the **search box** at the top of the dashboard to instantly filter the project list. Search matches against:

- Project name
- Framework (e.g., "Next.js", "Express")
- ORM (e.g., "Prisma")
- Project slug

Search is case-insensitive. Press **`/`** from anywhere on the dashboard to jump to the search box.

## Filtering by Status

Use the filter buttons below the search box to show only projects with a specific status:

| Filter | What it shows |
|--------|---------------|
| **All** | Every project (default) |
| **Active** | Projects you're actively working on |
| **Paused** | Projects temporarily on hold |
| **Archived** | Projects no longer in development |

Only one filter can be active at a time.

## Sorting

Use the sort dropdown to change how projects are ordered:

| Sort option | Description |
|-------------|-------------|
| **Last Activity** | Most recently changed projects first (default) |
| **Last Claude Session** | Projects you most recently used Claude with |
| **Name** | Alphabetical order |
