# Command Palette

Press **Ctrl+K** (or **⌘K** on Mac) from any page to open the command palette. It lets you navigate to any route, project, session, or agent without using the nav menu.

## What's in the palette

- **All navigation routes** — every page in the app (Dashboard, Sessions, Settings, etc.)
- **Projects** — all scanned projects in your dev root, with their path as a sublabel
- **Recent sessions** — the 50 most recent Claude Code sessions
- **Agents** — all indexed agents across your projects and plugins
- **Quick actions** — Emergency Stop and other one-click actions

## Keyboard navigation

| Key | Action |
|-----|--------|
| `↑` / `↓` | Move through results |
| `Enter` | Select highlighted item |
| `Escape` | Close palette |

## Recent selections

Items you've selected recently appear at the top when the query is empty. Up to 10 recents are stored in `localStorage` under `minder.palette.recent`.

## Changing the shortcut

Go to **Settings → Appearance → Keyboard shortcuts** and click **Edit** next to "Command palette" to assign a different combo.
