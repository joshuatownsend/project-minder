# Insights Report

The **Insights Report** page (`/insights-report`) displays the Claude Code usage-analysis report generated by `claude /insights`.

## Generating the report

The report is not generated automatically. Run the following command in any Claude Code session:

```
/insights
```

This creates `~/.claude/usage-data/report.html`. Reload the page after running it.

## What the report shows

The report is generated by Claude Code itself and contains session statistics, token usage, tool patterns, and productivity trends over your coding history.

## Where to find it

**Sessions → Insights Report** in the nav, or press **Ctrl+K** and type "Insights Report".

## Regeneration

Automatic regeneration from the dashboard is not available — `claude /insights` is an interactive REPL command and cannot be invoked headlessly. Run it manually and reload.
